# گالری آنلاین طلا و جواهر [Galdin]

طراحی و پیاده‌سازی پلتفرم اختصاصی فروش آنلاین طلا با تمرکز بر **تجربه کاربری لوکس (Premium UI)** و **امنیت تراکنش‌های مالی**.

## 💎 درباره پروژه
صنعت طلا و جواهر به دلیل ارزش بالای کالا و نوسانات قیمت، نیازمند سایتی است که هم بسیار سریع باشد و هم اعتماد کاربر را جلب کند. این پروژه با استفاده از معماری وردپرس به گونه‌ای بهینه‌سازی شده که علاوه بر سرعت بالا در بارگذاری تصاویر محصولات، فرآیند خرید را برای مشتریان بسیار ساده و امن کرده است.

## ✨ چالش‌های فنی و دستاوردها
- **نمایش بصری قدرتمند:** پیاده‌سازی گالری‌های حرفه‌ای که جزئیات و ظرافت‌های جواهرات را با کیفیت بالا نمایش می‌دهد.
- **اعتمادسازی (Trust Building):** طراحی رابط کاربری (UI) متمرکز بر حسِ لوکس بودن و امنیت، جهت افزایش نرخ تبدیل بازدیدکننده به خریدار.
- **امنیت:** پیاده‌سازی استانداردهای حفاظتی برای جلوگیری از بروز خطا در درگاه‌های بانکی و حفاظت از دیتای کاربران.

## ⚙️ ابزارهای استفاده شده
- **هسته:** WordPress & WooCommerce
- **طراحی: Elementor  
- **بهینه‌سازی:** پیاده‌سازی کشینگ حرفه‌ای و فشرده‌سازی تصاویر برای لود سریع سایت
